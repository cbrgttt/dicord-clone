# Документация к базовому проекту defaultWebpack

### Структура проекта

```
src
------fonts - папка со шрифтами
------images
------------pictures - фотографии
------------icons - иконки
------------webp - хранилище картинок в формате .webp
------------favicon.ico - иконка сайта
------js
------------events - дирректория для файлов с навешиванием ивентов
------------helpers - вспомогательные функции&классы
------------------classes
------------------------types - типы вспомогательных классов
------------------functions
------------services - сервисы
------------index.js - entry point
------scss
------------libs - вспомогательные сборники классов
------------_index.scss - выходная точка классов
```
